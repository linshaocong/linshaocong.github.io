    一. Hexo主题选择

    Hexo的主题有很多，大家可以访问 https://hexo.io/themes/ 来挑选，点击图片可以看到demo，部分需要科学上网，选择一个你喜欢的，现在主流的是用Next主题，我们这里将会用Anatole 主题来做演示，后续也会写Anatole的细节修改步骤；

    1. 打开 主题选择 的页面，我们找到Anatole ，然后点击图片下方的字进入到这个库；

    2. 点击clone，复制他的库地址；
